##Pre-requisites
- bower
- angular.js
- bootstrap
- jquery
- kendo-ui

first install bower (note: node.js needs to be already installed):

`npm install -g bower`

Then, inside the source folder, you can install the needed packages:

`bower install angular`

`bower install bootstrap`

`bower install kendo-ui`
